About Us

This app works as a trainer and teaches several pre-programmed poses like mountain,tree, downdog and warrior. After giving them a brief about the Exercise and its benefits (optional for the user), the user can start doing their yoga session. YogNMe will track the performance of the user using the mobile camera. A timer for the exercise will automatically start and will track the progress of the user. If a user moves into the wrong posture/pose for the exercise which reduces its benefits or even causes any type of injuries, YogNMe will warn the user and give the user options like, restart session, learn again, quit or skip exercise.
